# ── BROWSER REQUIREMENTS ────────────────────────────────────
#  The following browser APIs must be supported.
#  All modern browsers (2020+) support these natively.

#   Canvas API (2D context)         — animations, QR, photon channel
#   requestAnimationFrame           — 60fps canvas rendering
#   setInterval / setTimeout        — photon spawning, key rotation
#   ES6+ JavaScript                 — arrow functions, destructuring,
#                                     template literals, Array.from,
#                                     spread operator, let/const

#   CSS Grid & Flexbox              — layout
#   Google Fonts API                — web font loading
